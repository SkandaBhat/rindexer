# Chatbots

:::info
rindexer Chatbots can be used without any other storage providers. It can also be used with storage providers.
:::

rindexer has first-class support for Chatbots, this means you can use your favourite chat platform to
send messages to when events happen on chain.

:::info
Due to rate limits and also what most people require rindexer Chatbots will only start sending messages with a largest
block range of 10 blocks. Telegrams bots are really only meant to be ran sending live data not historic data.
:::

Supported Chatbots providers:

- [Telegram](/docs/start-building/chat-ops/telegram) - Send messages to your Telegram chats
- Discord - Coming soon
- Slack - Coming soon
