# Project Types

rindexer has two types of projects you can create:

- [No-code Project](/docs/start-building/project-types/no-code-project) - Short summary: No-code project is a project where you can start indexing without writing any code.
- [Rust Project](/docs/start-building/project-types/rust-project) - Short summary: Rust project is a project where you can create a custom and more advanced indexing systems using Rust.
